# Activation Guidelines


This skill should be activated when:
- Designing RESTful or GraphQL APIs
- Planning database schemas
- Building microservices architectures
- Implementing authentication/authorization
- Designing caching strategies
- Planning for scalability
- Implementing message queues
- Creating API documentation
- Designing service-to-service communication
- Planning deployment architectures
- Implementing observability solutions

The skill provides the most value when given clear requirements, constraints, and expected scale.
