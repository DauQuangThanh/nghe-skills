# Core Capabilities


When activated, this skill enables you to:

1. **Test Strategy & Planning**
   - Define integration testing scope and objectives
   - Identify integration points and dependencies
   - Create test plans and test cases
   - Design test data management strategies
   - Plan test environments and infrastructure

2. **API Testing**
   - Test RESTful API endpoints
   - Test GraphQL queries and mutations
   - Validate request/response schemas
   - Test authentication and authorization
   - Implement contract testing

3. **Database Integration Testing**
   - Test database queries and transactions
   - Validate data integrity and constraints
   - Test migrations and schema changes
   - Implement test data fixtures
   - Test database performance

4. **Microservices Testing**
   - Test inter-service communication
   - Implement service contract testing
   - Test message queues and event streaming
   - Validate distributed transactions
   - Test service discovery and load balancing

5. **End-to-End Testing**
   - Design user journey tests
   - Test complete business workflows
   - Implement UI-driven integration tests
   - Test cross-platform functionality
   - Validate system integration

6. **Test Automation**
   - Select appropriate testing frameworks
   - Implement test automation pipelines
   - Integrate with CI/CD systems
   - Generate test reports and coverage
   - Implement parallel test execution
